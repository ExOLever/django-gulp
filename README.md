Django Gulp Example
=============

Django gulp example

## Install
    $ mkvirtualenv <virtualname>
    $ pip install -r requeriments.txt
    $ cd example/frontend
    $ npm install
    $ bower install
    $ gulp dev / gulp build
    $ cd ..
    $ python manage.py migrate
    $ python manage.py runserver
